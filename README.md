======================================

This is a framework independent workflow library which is used for the framework extension for the CMS Contao.

It provides an step-transition based workflow implementation for processing entities through its life cycle.

As this is a framework independent solution you have to implement some parts so it gets usable.


Credits
-------

This library is heavenly inspired by the great workflow implementation of [orocrm plattform](http://github.com/orocrm/plattform)
and got some concepts from the [LexikWorkflowBundle](https://github.com/lexik/LexikWorkflowBundle).
